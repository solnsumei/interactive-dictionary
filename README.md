# Interactive Dictionary
A console based interactive dictionary built with python with a JSON data source for word definitions

## Features
- Up to date english dictionary
- Word suggestions
- Acronyms inclusive

## Installation
- Install python 3+

## Usage
- From your terminal
- Navigate to the project folder
- Run
```
python app.py
```

## Test
```
pip install pytest
run pytest
```
